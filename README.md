# Dokumentasi Kode HTML

Berikut adalah dokumentasi untuk kode HTML yang digunakan untuk membuat tabel nilai siswa:

```html
<!DOCTYPE html> <!-- Mendefinisikan dokumen sebagai dokumen HTML5 -->
<html lang="en"> <!-- Elemen root dari halaman HTML -->
<head> <!-- Mengandung metadata tentang dokumen HTML -->
    <meta charset="UTF-8"> <!-- Menentukan set karakter yang digunakan oleh halaman -->
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/modern-normalize@2.0.0/modern-normalize.min.css"> <!-- Link ke CSS Normalize -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- Membuat halaman responsif -->
    <title>tugas 1 table</title> <!-- Judul halaman web -->
    <style> <!-- Style internal untuk halaman -->
        /* ... */
    </style>
</head>
<body> <!-- Bagian utama halaman web -->
    <table width="25%" align="center"> <!-- Tabel dengan lebar 25% dan rata tengah -->
        <!-- Baris dan kolom tabel -->
    </table>
</body>
</html>
